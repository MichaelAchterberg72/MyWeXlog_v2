Add a New Book Page - Help

 

This page displays the fields that are required to capture a new book.

The fields required are:

Book Title Enter the title of the book.
Type Select from the drop-down list wither Fiction (describes imaginary events and people) or Non-Fiction (informative or factual).
Publisher Select from the drop-down list the publisher by typing the publisher name into the search field at the top of the dropdown list.
The Add a new Publisher link will open a popup window and request you to add the publisher name and the publisher url.
Author Select from the drop-down list the author by typing the author name into the search field at the top of the dropdown list. The author names are added in the form of tags to allow you to enter multiple names.
The Add a new Author link will request you to add the Author Name if your desired option is not in the list.
Genre Select from the drop-down list the genre by typing the enre type into the search field at the top of the dropdown list. The genre types are added in the form of tags to allow you to enter multiple entries.
The Add a Genre link will request you to add the Genre Type if your desired option is not in the list.
Book URL Please enter the URL if available for the book should someone wish to link to it.
Tag / Associated Skill Allow you to select the skills you have learned or developed whilst reading from or learning from the book. Select from the dropdown list and will be listed as tags that you can delete is so wished. You are able to capture as many skill tags as are relevant.
To read a description of how to capture skills please click here.
The Add a new Skill Tag link will allow you to capture a new skill into the system. When you save a new skill it will automatically be populated in the Skills field.

**In the event that one of the dropdown lists is not populated please refresh the page.

A * next to the field name indicates that the field is a required field.

Click the button when complete.
