Booklist Home Page - Help

 

This page displays a list of the books that you have captured as having read.

The books are listed in order of having been read and displayed with the book name (publisher), the book type and the date read. This information is displayed in your public profile under the heading of Books Read. The profile will also list the Total Books Read: displaying them by Title, Publisher, Author(s) Format and Date Completed. If you have not captured any books it will state "No books listed".

You have the option to:

  • Add Book - Button will allows you to capture a new book read requiring the Book to be selected from the drop down list, Book Format and Date that you finished reading the book.
  • View All Books - Button will display a paginated listing in alphabetical order of all the books that have been captured into the MyWeXlog system.
  • Search All Books - Button will allow you to search the books captured in the MyWeXlog system. The search function all you to search based on the title, skills, author and publisher.
  • Book Detail - will open the detail page for the book with Title, Type, Publisher, Author(s), Skills and url.
