Books Search Page - Help

 

This page displayes a list of the books that are captured in the WeXlog system that meet your keyword parameter.

The search function will search against the Title, Skills Learned, Author and Publisher.

The books are listed in alphabetical order by Title.

The Book Detail button will open the Details listing for the book.

You have the option to:

  • Add Book - Button will allows you to capture a new book requiring the Book Title, Type, Publisher, Author, Genre, url and skill tags.
  • View All Books - Button will display a paginated listng in alphabetical order of all the books that have been captured into the WeXlog system.
  • Search All Books - Button will allow you to search the books captured in the WeXlog system. The search function alls you to searh based on the title, skills, author and publisher.
  • Book Detail - will open the detail page for the book with Title, Type, Publisher, Author(s), Skills and url.

The Search Again button wil refreash the search page to the initial search form requesting you to enter your search keyword.

You have the option to:

  • Back - allows you to navigate back to the Booklist Home page.
