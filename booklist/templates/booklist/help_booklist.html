Booklist Page - Help

 

This page displayes a list of all the books that you have captured in the WeXlog system.

The books are listed in alphabetical order by title listing the Book Name and Publisher. If you have not captured any books it will state "No books listed".

You have the option to:

  • Add Book - Button will allows you to capture a new book requiring the Book Title, Type, Publisher, Author, Genre, url and skill tags.
  • View Books Read - Button will display a paginated listng in datel order, most recent first, of all the books that you have read.
  • Search All Books - Button will allow you to search the books captured in the WeXlog system. The search function alls you to searh based on the title, skills, author and publisher.
  • Book Detail - will open the detail page for the book with Title, Type, Publisher, Author(s), Skills and url.

You have the option to:

  • Back - allows you to navigate back to the Booklist Home page.
