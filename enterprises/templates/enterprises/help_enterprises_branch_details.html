Add a New Enterprise Branch Details Page - Help

 

This page displays the fields that are required to capture a new Enterprise Branch.

The fields required are:

Company This will be selected from the dropdown list and should be the main company that oversees the branch.
The Add a Company link will request you to add brief details of the main company being the Enterprise name, the website address and a description of the company.
Branch or Division Name This should be descriptive of the branch so it is easily identifiable by people that are in collaboration companies as well.
Type This will be selected from the dropdown list and should be descriptive of the branch type be it a Head Office, Branch Office or Satellite Office.
The Add a new Branch Type link will request you to add a branch type.
Physical Address This will allow you to capture the address of the offices allowing for an Office Block address and street address.
Industry This will be selected from the dropdown list and should be the main industry in which the company that is controlling the project operates.
The Add a new Industry link will request you to add an Industry Type.
Country This will be selected from the pre-populated dropdown list of all countries.
Region This will be selected from the dropdown list and should list all regions in the selected country.
The Add a Region link will request you to add an new Region requiring the Country and Region / State / etc.
City This will be selected from the dropdown list and should list all cities in the selected Region.
The Add a City link will request you to add a new City requiring the Region and City, Town or Place.
Suburb This will be selected from the dropdown list and should list all suburbs in the selected City.
The Add a Suburb link will request you to add a new City requiring the Region and City, Town or Place.
Post Code This will allow you to capture the Physical Address Postal Code and is a required field.

**In the event that one of the dropdown lists is not populated please refresh the page.

A * next to the field name indicates that the field is a required field.
