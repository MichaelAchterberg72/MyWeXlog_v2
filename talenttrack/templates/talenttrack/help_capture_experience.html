Capture Experience - Help

 

This page displays the fields that are required to capture your Work Experience.

The fields required are:

Company You are able to type the name of the company into the dropdown list search field. THe list will only show the company, not the branch.
The Add a Company link will request you to add a new company details such as the enterprise name, the website as well as description of the company.
Company Branch This will be filtered by the selected company and selected from the dropdown list and will be the company name followed by the branch name and city.
The Add a Branch link will request you to add details of the branch, this being the company name, branch or division name, branch type and the physical address.
On Project This will be selected from the dropdown list and will be the company name followed by the project name and city.
The Add a Project link will request you to add details of the project name, the project owner, project site, industry and regional address information.
Date from This will be input utilising the date widget that is accessible from the date widget that appears when you hover the mouse over the field. By clicking on the arrows on the right hand side of the date field and clicking them clicking on the year you can navigate to the date you require and you can select the date required.
Date to This works in the same way as the date from field but should be the date you have completed the course subject.
Hours Worked This should be the number of hours actually worked between the start and end dates of the capture period.
Estimated This should be selected if the time captured is more than two weeks old.
Industry This will be selected from the dropdown list and should be descriptive of the industry in which the project operates.
The Add an Industry link will request you to add the Industry type.
Designation This will be selected from the dropdown list and should be the designation such as manager or director.
The Add an Designation link will request you to add a new designation type.
Upload This allows you to upload a document such as a timesheet or certificate of performance.
Skills This will allow you to select the skills you have utilised in the performance of your work during the capture period from the dropdown list and will be listed as tags that you can delete is so wished. You are able to capture as many skill tags as are relevant.
To read a description of how to capture skills please click here.
The Add a Skill link will allow you to capture a new skill into the system. When you save a new skill it will automatically be populated in the Skills field.
Comment This allows you to capture a description of the work performed along with any comments you feel would be relevant to document the experiences in the time period captured.

**In the event that one of the dropdown lists is not populated please refresh the page.

A * next to the field name indicates that the field is a required field.

You have the option to:

  • Add Validators - allows you to navigate to the page which allows you to capture Colleagues, Superiors, Collaborators and Clients that will be requested to validate your experience, skills learnt and time.
    If you do not want to select any Colleagues, Superiors, Collaborators or Clients click on the 'Skip' button whilst working through the validation wizard.
