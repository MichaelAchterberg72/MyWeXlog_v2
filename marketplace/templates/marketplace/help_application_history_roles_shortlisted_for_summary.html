Vacancies Application History Page - Roles Shortlisted For - Help

 

This provides a brief list of the ten most recent vacancies that you have been shortlisted for. This will include the positions where you have been selected from a searh list as having the skills required for the position as well as roles that you have submitted an application for the vacancy that you have been selected for consideration shortlisting.


The table lists the date you applied for the vacancy, the vacancy title, the system reference number, the hours required by the position, whether or not the position is a local or a remote work configuration, the city that the work office is provided from and the status of the position as well as a button for opening the full detail of the vacancy where you will be able to apply for the role.

The options you will be provided with when on the table will be:

Opens the detail page of the vacancy allowing you to apply for the vacancy should it still be open. On the detail page, if the vacancy is closed it will show a closed button and you will not be able to apply for the position.
See Detail Should you not have an Active Subscription you will be provided with an inactive 'See Detail' link.


The following buttons will only be visible if you have an Active Subscription

View Full List

View Full List

This will open the page listing the paginated full list of vacancies that you have been shortlisted for with the same information and functionality as described above.
