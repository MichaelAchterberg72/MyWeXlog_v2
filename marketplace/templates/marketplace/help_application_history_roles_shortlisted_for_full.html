Shortlisted For Full List - Help

 

This provides a paginated full list of the vacancies that you have been shortlisted for. This will include the positions where you have been selected from a search list as having the skills required for the position as well as roles that you have submitted an application for the vacancy that you have been selected for consideration shortlisting.


Back Button

Take you back to the Vacancy Application History summary page.



The table lists the date you applied for the vacancy, the vacancy title, the system reference number, the hours required by the position, whether or not the position is a local or a remote work configuration, the city that the work office is provided from and the status of the position as well as a button for opening the full detail of the vacancy where you will be able to apply for the role.

The options you will be provided with when on the table will be:
