Vacancy Review - Help

 

This page displays the information about the vacancy allowing you to add, edit or remove the deliverables and skills. It will provide a search feature of all Talent suited to the Vacancy as well as listing the Applicants that have submitted a bid for the vacancy. Talent or applicants may be shortlisted from these lists and a button is available to open the Short-list as well as the interview list.


Navigation Buttons

Open the page listing the paginated full list of vacancies that you have been accepted from after having been shortlisted for with the same information and functionality as described above.

The button options available are:

Navigate you back to the vacancies home page.
Edit Vacancy Open the vacancy in a form allowing you to edit as required.
View Short-list Open the short list for the vacancy listing all talent that has been selected for short listing.
View Interview List Detail the short listed talent that has been selected for an interview.
Enter Office Navigate to the Assignments Listing page which breaks the listing into a listing across all vacancies for "Open Assignments", "Pending Assignments" and "Completed Assignments".
Interview History Detail listing of all interviews scheduled across all Vacancies.


Deliverables

Allow you to add to, edit or remove the listed deliverables as depicted.


Skills

Allow you to add to or remove the list of skills as depicted.
