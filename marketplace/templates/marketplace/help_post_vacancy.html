Add or Edit a Vacancy - Help

 

This page displays the fields that are required to capture a new or edit vacancy.


Example - How to capture a full time position

Capture all the fields as required and stipulate in the "Hours" as to the expected number of hours required and then in the "Unit" selector stipulate "per Week" or "per Month" as the position may require.

Rember to ensure that you have populated the "Brief Career History" section of the profile with a current position to ensure that the "Companybranch" field will have your current company that you are posting the position for available in the drop down list.


The fields required are:

Vacancy Title

Enter the title for the vacancy making sure that it is reflective of the vacancy and position that it refers to.

Vacancy Reference Number

A unique reference number for the vacancy. This is the reference number that will be used in the event of performing a search.

Company Branch

Type the company name in the search field at the top of the dropdown list that appears when you click on the field, select from the dropdown list the Company name and branch that you are wanting, if you add a new Company Branch it will automatically populate the field.
The Add a Company Branch link will open a popup window and request you to add the company, branch or division name, type, and physical address.

Hours Enter the number of hours the vacancy is expected to take for completion or alternatively the number of hours the company has budgeted for at the rate offered.
  • For a permanent position, enter the number of hours an employee is expected to work each month (typically this is 163 hours per month). The "unit" field should then be set to "per month".
Unit The manner in which the estimated hours for the vacancy are intended to be spent. The selection possibilities are:
  • once off
  • per day
  • per week
  • per month: used for full time employees or a long term contract position.
  • short term contract
  • as and when contract
Experience Level Select from the drop down list the experience level required for prospective tenders on the vacancy. The categories are:
  • Breakdown of catagories:
Currency

Select the currency that you will be remunerating with from the dropdown list and if you add a currency that is not available ion the list it will automatically populate the field.
The Add a Currency link will open a popup window and request you to add the country, currency in full written format with country and currency name and the internationally recognised currency abbreviation like USD or AUD.

Rate Offered

This is the rate you are willing to offer for the work advertised. This rate must be entered in conjunction with the "Rate Unit" field.

Rate Unit This is how the rate is intended to be paid, not when payment will be made. The options are:
  • per Hour
  • per Day
  • per Month
  • Lump Sum
Configuration How or where the candidate is expected to work. The options are:
  • Breakdown of configerations:
City, Town or Place

Select from the dropdown list and the localised location the company branch operates.
The Add a City, Town or place link will request you to add the Region and City, Town or Place name if your desired option is not in the list.

To be Completed By

Here please enter the date that you required the job to be completed. Open the date picker by clicking on the up or down arrows that appear on the right of the capture field when you hover over it.

Vacancy Status Select whether or not the vacancy is either:
  • Open
  • Closed
Vacancy Closes

Here please enter the date that vacancy offering for the job to be completed. Open the date picker by clicking on the up or down arrows that appear on the right of the capture field when you hover over it.

Certifications Required

Please enter any certifications that will be required for the performance of the vacancy. You may start typing the name of the certificate and the search field of the drop down list will filter for what you are looking for and capture the required certificates in tags allowing more than one to be captured.
The Add a Certificate link will request you to add a certificate type if your desired option is not in the list.

Scope

Here you are able to enter the scope for the vacancy offering to be completed. The field will allow you to enter paragraphs.

Expectations

Here you are able to enter the expectations for the vacancy offering to be completed. The field will allow you to enter paragraphs.


**In the event that one of the dropdown lists is not populated please refresh the page.

A * next to the field name indicate that it is a required field.
