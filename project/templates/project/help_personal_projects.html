Personal Projects Active Page - Help

 

This page displays a list of the projects that are captured in the MyWeXlog system thast you have captured as being personally active in.

The projects are listed in alphabetical creating a listing of each project listed by the Company and the Branch specific order detailing the Project Name, Project Industry and the project city.

The Personal
Detail button will open the project personal details page which will detail the Project Owner, Project Name and the project company you are working for as listed in MyWeXlog. You will be able to view and amend the description you have captured specifying your personal responsibilities in the project.

The See Project
Detail button will open the project details page which will detail the Project Owner, Project Name, Industry, Total Project Hours, Country, City, Region and the Total number of people on the project as listed in MyWeXlog.

You have the option to:

  • Add Project - Add a new project which will request you to supply the Project Name, Project Owner, Project Site, Industry, Country, Region and a City.
  • Full Project List - View a listing of all projects listed in MyWeXlog detailing the Project Name, Project Owner, Industry, Country City.
  • Search Projects - Using a Keyword search the projects based on the Project Name, Company Name, Industry, Country, Region and City.
  • Back - allows you to go back to the Enterprises Home page.
