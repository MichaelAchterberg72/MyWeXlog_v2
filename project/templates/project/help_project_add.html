Add a New Project Page - Help

 

This page displays the fields that are required to capture a new Project.

The fields required are:

Project Name This should be descriptive of the project so it is easily identifiable by people that are in collaboration companies as well.
Project Owner This will be selected from the dropdown list and should be the main company that is controlling the project.
The Add a new Enterprise link will request you to add brief details of the main company being the Enterprise name, the website address and a description of the company.
Project Site This will be selected from the dropdown list and should be the branch or site details of the company that is controlling the project.
The Add a Branch link will request you to add the Main Company Name, the Branch or Division Name, the Branch Type as well as the physical address of the branch.
Industry This will be selected from the dropdown list and should be the main industry in which the company that is controlling the project operates.
The Add a new Industry link will request you to add an Industry Type.
Country This will be selected from the pre-populated dropdown list of all countries.
Region This will be selected from the dropdown list and should list all regions in the selected country.
The Add a new Region link will request you to add an new Region requiring the Country and Region / State / etc.
City This will be selected from the dropdown list and should list all cities in the selected Region.
The Add a new City link will request you to add a new City requiring the Region and City, Town or Place.

**In the event that one of the dropdown lists is not populated please refresh the page.

A * next to the field name indicates that the field is a required field.

You have the option to:

  • Back - allows you to navigate back to the Projects Home page.
