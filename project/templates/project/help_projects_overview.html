Projects Overview - Help

 

The projects app is designed to provide a listing of projects, both a full list allowing you to view the information captured within MyWeXlog detailing the owner, companies involved, country, region and city the project is active in as well as the number of people and hours captured relating to the project and a listing of the project you are personally involved in.

The projects are listing of projects you are personally involved in allows you to view the persoanl information relating to that projects that you have captured within MyWeXlog.

You have the option to:

  • Add Project - Add a new project which will request you to supply the Project Name, Project Owner, Project Site, Industry, Country, Region and a City.
  • Project Home - View a listing of all projects listed in MyWeXlog personally involved in detailing the Project Name, Project Owner, Industry, Country City.
  • Full Project List - View a listing of all projects listed in MyWeXlog detailing the Project Name, Project Owner, Industry, Country City.
  • Search Projects - Using a Keyword search the projects based on the Project Name, Company Name, Industry, Country, Region and City.
  • Back - allows you to go back to the Enterprises Home page.
